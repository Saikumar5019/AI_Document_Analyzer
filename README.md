# AI Document Analyzer

AI Document Analyzer is an intelligent document processing application that allows users to upload documents and automatically extract meaningful insights using OCR and local LLMs.

The system can process PDFs, DOCX files, TXT files, and image-based documents, classify document types, generate summaries, and extract structured entities — all running locally using Ollama without requiring paid API keys.

---

## Features

- Upload and analyze multiple document formats
- PDF text extraction
- DOCX parsing
- TXT file parsing
- OCR support for image documents (PNG, JPG, JPEG)
- AI-powered document classification
- AI-generated document summarization
- Structured entity extraction in JSON format
- Fully local LLM inference using Ollama
- Streamlit-based interactive UI
- No external API dependency

## Installation

### 1. Clone Repository

```bash
git clone https://github.com/your-username/AI_Document_Analyzer.git
cd AI_Document_Analyzer
```

---

### 2. Create Virtual Environment

Windows:

```bash
python -m venv venv
venv\Scripts\activate
```

Linux / Mac:

```bash
python3 -m venv venv
source venv/bin/activate
```

---

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 4. Install Ollama

Download Ollama:

https://ollama.com/download

Install the required model:

```bash
ollama pull llama3.2
```

Verify:

```bash
ollama list
```

---

### 5. Run Streamlit Application

```bash
streamlit run app/main.py
```

## Usage

1. Launch the Streamlit app
2. Upload a supported document
3. System extracts text using parser or OCR
4. AI model classifies document type
5. Summary is generated
6. Structured entities are extracted
7. Results are displayed in UI
